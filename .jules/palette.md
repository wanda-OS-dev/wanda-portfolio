## 2024-05-24 - Explicit Copy Feedback
**Learning:** Subtle icon changes or relying on native `title` attributes (e.g. changing from "Copy" to "Copied!") is often insufficient for immediate visual confirmation, especially when interacting rapidly. Screen readers get the feedback via `aria-live`, but sighted users might miss a subtle SVG change.
**Action:** When implementing utility actions like 'Copy to clipboard' that do not trigger native browser alerts, provide explicit, temporary text-based visual feedback (e.g., an animated 'Copied!' span) adjacent to the trigger. Ensure this visual element has `aria-hidden="true"` to prevent duplicate announcements for screen reader users if an invisible `aria-live` region is already handling the announcement.
