# wanda-portfolio

> Wanda AI Portfolio — Next.js 14 + React Three Fiber.

A high-performance portfolio showcasing **WandaSystems**' AI capabilities and 3D visual storytelling. Built for speed, immersion, and autonomous updates.

## ⚡ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **3D Engine**: [Three.js](https://threejs.org/) / [@react-three/fiber](https://github.com/pmndrs/react-three-fiber)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)

## 🏗️ Architecture

- **React Three Fiber (R3F)**: Immersive 3D background and interactive scene.
- **Next.js App Router**: Server-side rendering for critical content, client-side hydration for interactions.
- **Responsive Layout**: Optimized for mobile touch interfaces and desktop interactions.

## 🛠️ Development

### Setup

```bash
# Clone the repository
git clone https://github.com/wanda-OS-dev/wanda-portfolio.git

# Install dependencies
npm install

# Run dev server
npm run dev
```

### Production Build

```bash
npm run build
```

The output will be optimized for Vercel or custom Node.js hosting.
